## Présentation
Nous sommes deux lycéens qui à un moment voulaient créer une application pour intégrer pleins d'outils pour gérer son PC, installer des applications les mettres à jours ou les désinstallers, et pleins d'autre outil qu'on a en tête qui s'ajoutera.

## Pourquoi ce logiciel ?
Keera Tools est là pour permettre de lier pleins d'utiliser en un seul et même utilitaires, Gratuit et Open-Source.

## Le logiciel est fait comment ?
Keera Tools est fait entièrement en Python, cela facilite la tâche pour nous et est ultra-facile grâce à son nombre incalculable de librairies disponibles dans ce langage. D'autre language peuvent être utiliser pour les outils futurs.

Pour la version Windows, le logiciel utilise l'outil Winget pour installer des applications via Powershell (pour information, si vous venez d'installer Windows et que vous utilisez le logiciel, il vous demandera de le mettre à jour). Pour la personnalisation elle utilise principalement le powershell et le regedit.


Le logiciel est fonctionnel, mais il reste quelques soucis côté graphique ou utilisation du logiciel des Patch viendrons petit à petit mais en tout cas si vous trouver des bug ou des instabilité ou des fonctionnalité qui ne marche pas faite nous signe et on envera une mise à jours pour les corriger.

## Comment utiliser le logiciel ? 
Vous avez deux option :
1. Installer le logiciel : Vous installer le logiciel et il sera utilisable dans votre pc juste à rechercher dans la recherche Windows KeeraTools et vous l'aviez.
   
2. Version Portable : Le logiciel a une version portable qui permet d'être utiliser par tout y compris une clef usb il faut juste double clique dessus et il se lance.
