## Bienvenu sur la page Github de Nova Installer.

Nova Installer est un proget indépendant de 2 gars qui s'ennuiaient donc ils ont codé une appli avec ChatGPT pour télécharger des applis.\
La version Linux utilise Flatpak (une version DNF/APT/PACMAN/etc) est en cours de réflection.
La version Windows utilise Winget/Chocoltey.

Notre code source (de chaque version) ets trouvable dans les répertoires.
